🍎 EduSupply Chain – Gestão Inteligente de Inventário Escolar
O EduSupply Chain é uma plataforma SaaS concebida para transformar a gestão de suprimentos e alimentação em instituições de ensino. O foco central do projeto é a eliminação do desperdício de alimentos e a eficiência na distribuição de materiais escolares através de um controle de inventário rigoroso e preditivo.

🌟 Diferenciais do Projeto
Monitoramento em Tempo Real: Gestão de entradas e saídas de stock com alertas de validade.
Redução de Desperdício: Algoritmos para cálculo de demanda baseados no consumo real das instituições.
Escalabilidade em Nuvem: Implementação completa utilizando Docker e Google Cloud Run, garantindo alta disponibilidade.
Interface Responsiva: Desenvolvida para ser utilizada tanto em ambientes administrativos quanto em operações de campo (cozinhas e depósitos).

🛠️ Stack Tecnológica
Frontend: Next.js (React) com foco em Server-Side Rendering (SSR) para performance.
Backend: Node.js para processamento de regras de negócio.
Infraestrutura: Google Cloud Platform (GCP), Cloud Run e Docker.
Base de Dados: MongoDB.

## Run Locally
**Prerequisites:**  Node.js
1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
